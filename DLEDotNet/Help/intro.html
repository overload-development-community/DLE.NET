Creating a new level

Create a new level by clicking "New..." from the File menu. The new level dialog asks for the level name and type. The type options are.

  • Descent 1
  • Descent 2
  • D2 Vertigo (Descent 2 Vertigo, required to use new robot types introduced in Vertigo)
  • D2X-XL (extra features, only playable in D2X-XL)
A level can be converted later, but most conversions cannot be done without hassle. The general rule is that in the above list, converting to one of the types below is easy, while converting to one of the types above is difficult.
