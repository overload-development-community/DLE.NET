DLE.NET - Descent Level Editor .NET

DLE.NET is a level and mission editor for Descent (1), Descent 2 and D2X-XL.

It is an improved .NET port of DLE-XP by Dietfrid Mali, which is itself based on Descent Mission Builder 2 by Bryan Aamot of Brainware.

In order to use DLE.NET to its fullest potential, you need to own a full copy of Descent and/or Descent 2.

Controls

Controls are documented on their various pages and are also available by browsing through the menus. Note that any controls mentioned on these help pages are the default controls for those actions, which will eventually be customizable.

Creating a new level

Create a new level by clicking "New..." from the File menu. The new level dialog asks for the level name and type. The type options are.

  • Descent 1
  • Descent 2
  • D2 Vertigo (Descent 2 Vertigo, required to use new robot types introduced in Vertigo)
  • D2X-XL (extra features, only playable in D2X-XL)
A level can be converted later, but most conversions cannot be done without hassle. The general rule is that in the above list, converting to one of the types below is easy, while converting to one of the types above is difficult.
