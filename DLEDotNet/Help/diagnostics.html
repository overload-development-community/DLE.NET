Diagnostic Tool

Statistics · Problem Report

This tool can be used to check for problems in the mine that can affect the playability and stability of the mine while played in-game. It also contains statistics for the level.

Statistics

Text Description
segments The number of segments.
vertices The number of points, where two joined points are considered as one.
mat cens The number of robot maker segments.
fuel centers The number of energy center segments.
walls The number of walls.
triggers The number of triggers.
objects The number of objects, with further subcategories for different kinds of objects.
textures The number of unique textures used within the level.

Problem Report

The problem report is generated whenever the mine is checked by using the "check mine" option. "show problem" shows the segment or object that contains the problem, "clear list" clears the list, "auto fix bugs" controls whether problems should be automatically fixed on check if possible, and "show warnings" controls whether the problem list should contain warnings that may affect playability.
