Insert Menu

Text Description
Segment Adds a new segment pointing out from the current side. The menu selects the type of the segment to add; see the Segment Tool for more info.
Object Adds a new object to the current segment. The menu selects the type of the object to add; see the Object Tool for more info.
Door Adds a door on the current side. The door is added to both sides, and thus the side must be open. The menu selects the type of the object to add; see the Wall Tool for more info.
Trigger Adds a new trigger on the current side, including a wall if none is present. The menu selects the type of the trigger to add; see the Trigger Tool for more info.
Wall Adds a wall on the current side. The wall is added to both sides, and thus the side must be open. The menu selects the type of the wall to add; see the Wall Tool for more info.
