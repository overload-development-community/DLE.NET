Texture List

The texture list displays a grid of available textures (or the textures used in the mine if used textures is enabled), and allows picking textures for the selection.

The list below shows the controls:

Control Action
Left-click Set the currently selected texture as the 1st texture for the selected side.
Right-click Set the currently selected texture as the 2nd texture for the selected side.
Middle-click
Hold left, click right
Hold right, click left		 Finds the next segment/side with the given texture and selects it. If the texture is not used in the level, does nothing.
Shift + Left-click Set the currently selected texture as the overlay texture for the selected object (only works if the object is a robot or reactor).
Ctrl + Left-click Set the currently selected texture as the 1st texture for the trigger (only works for the "change texture" trigger type).
Ctrl + Right-click Set the currently selected texture as the 2nd texture for the trigger (only works for the "change texture" trigger type).
