How to browse Help

For browsing help, there are two distinct options:

  1. A frame-based setup that can be used here, if your browser (still) has frame support
  2. A basic table of contents / index page here that allows you to view a page one at a time

If you are using the former, you can switch to another page by using the sidebar on the left.

If you want to open a help page for an existing tool, you can right-click the tool on non-floating layouts or use the help button in the top-right corner of the tool on the floating layout.
