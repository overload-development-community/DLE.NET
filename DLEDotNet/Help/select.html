Select Menu

All of the next/previous actions will only work in the correct mode (for example, "Next object" requires the object mode), and if the correct mode isn't selected, it is selected instead. This means that using "next object" in side mode will switch to object mode without changing to the next object.

Text Description
Next object Selects the next object in sequence, or the first object if the last object is already selected.
Previous object Selects the previous object in sequence, or the last object if the first object is already selected.
Next segment Selects the next segment in sequence, or the first segment if the last segment is already selected.
Previous segment Selects the previous segment in sequence, or the last segment if the first segment is already selected.
Forward segment If the side is open, moves the selection to the neighboring segment; else moves the selection to one of the other neighboring segments, if any.
Backwards segment If the opposite side is open, moves the selection to that neighboring segment; else moves the selection to one of the other neighboring segments, if any.
Other segment Swaps the selections between the current and the other segment; the new current segment is the other segment and vice versa.
Next side Selects the next side of the current segment in sequence, or the first side if the last side is already selected.
Previous side Selects the previous side of the current segment in sequence, or the last side if the first side is already selected.
Next line Selects the next line of the current side in sequence, or the first line if the last line is already selected.
Previous line Selects the previous line of the current side in sequence, or the last line if the first line is already selected.
Next point Selects the next point of the current side in sequence, or the first point if the last point is already selected.
Previous point Selects the previous point of the current side in sequence, or the last point if the first point is already selected.
