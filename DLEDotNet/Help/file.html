File Menu

Text Description
New... Creates a new level. See the introduction page for more information.
Open... Opens an existing .HOG or level file.
Save Saves the current .HOG or level file.
Save As... Saves the current .HOG or level file under a new name.
Edit Mission File Opens the mission editor.
Edit Mission HOG... Opens the .HOG editor for the currently open .HOG file.
Edit Textures (POG)... Opens the .POG editor for the currently open level file.
Convert Mine to ... Allows converting the current level to another format.
Check Mine Opens the diagnostic tool and checks the mine for potential problems.
Run Level Runs the level in-game. Only D2X-XL supports this feature.
Preferences Opens the settings.
Information Opens the diagnostic tool, which contains level statistics.
recent files Displays the 10 last files to be opened or saved with the editor in order to make them quicker to open.
Exit Exits the program.
